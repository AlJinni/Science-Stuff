This folder contains the C++ header file, Constants.h.  In order to compile and link the other C++ codes for this text you will need to place Constants.h in the same folder as the code you wish to compile, or (preferably) you should place it in a folder that is accessible to your C++ compiler, such as an \include folder.

Constants is described in Appendix I of:

An Introduction to Modern Astrophysics
Bradley W. Carroll and Dale A. Ostlie
Addison Wesley
Copyright 2007.
